# Security And Privacy Rules (Strict)

## Core boundary
- This public repository is safe memory only.
- The private repository remains the full memory source.

## Never store in public
- API keys
- Tokens
- Passwords
- SSH keys
- Database credentials
- Private IP addresses
- Direct access commands
- Raw logs

## Publication policy
- Run a privacy scan before every push.
- If any content is uncertain, exclude it.
- Prefer safe summaries over operational details.
